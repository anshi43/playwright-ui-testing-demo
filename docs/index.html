QA Automation Portfolio Project

Playwright UI Testing Demo

A job-focused end-to-end test automation project built with Playwright and TypeScript. It demonstrates Page Object Model structure, smoke and regression coverage, shared authentication state, cross-browser execution, and CI integration.

GitHub Repository

Project highlights

  • Playwright with TypeScript
  • Smoke and regression test suites
  • Page Object Model implementation
  • Reusable auth state for logged-in flows
  • Cross-browser coverage: Chromium, Firefox, WebKit
  • HTML reporting and GitHub Actions CI

Covered scenarios

  • Valid and invalid login validation
  • Add product to cart
  • Cart verification
  • Product sorting by price and name
  • Full checkout flow
